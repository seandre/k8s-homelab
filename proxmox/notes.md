# Proxmox Notes

The 2 TB NVMe is installed and is the target disk for the final Proxmox VE build.

Attach the Proxmox host and Kubernetes VMs to the UniFi `Servers` network using the documented `192.168.40.x` addresses.

Use the 256 GB SSD history only as reference for dry-run lessons learned. Do not build long-lived services on the previous 256 GB layout.
