# Homelab
Kubernetes homelab built on Proxmox VE.
## Hardware
- Host: HP EliteDesk 800 G6 Mini
- CPU: Intel Core i5-10500T
- RAM: 64 GB
- Installed SSD: 2 TB NVMe
- Previous SSD: 256 GB
- Hypervisor: Proxmox VE
## Goal
Build a reproducible Kubernetes homelab for platform engineering practice.
Initial stack:
- Proxmox VE
- Ubuntu Server VMs
- k3s
- Argo CD
- ingress
- cert-manager
- monitoring
- local test apps
## Current Status
- 64 GB RAM installed
- 2 TB NVMe installed
- Ready for final Proxmox VE installation
- 256 GB SSD retired from the active build plan
